Projects I am working on

NATO Alphabet Challenge

This program was created as a game to help me get more familiar with the NATO alphabet. It is written in Java, and is structured to hold a string array, for codes (i.e. Alpha, Bravo, etc) and one for characters. I use a for loop to compare array positions to a user input.

New features planned for this project are:

  • GUI Interface
  • Typo catching - assigning values to keys surrounding the "Correct" key
  • Timer - start chasing a new high score!
